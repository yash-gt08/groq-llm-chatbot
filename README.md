🤖 Groq LLM Chatbot

A simple and interactive AI chatbot built using Streamlit and powered by Groq LLM API. This project provides a clean web interface to interact with a powerful Large Language Model.

🚀 Features

💬 Interactive chatbot interface
⚡ Ultra-fast responses using Groq LLM
 🌐 Web-based UI with Streamlit
